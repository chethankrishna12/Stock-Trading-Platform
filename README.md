# 🚀 Trading Platform

A full-stack stock trading platform inspired by real-world applications like Zerodha. This project demonstrates how modern trading platforms are structured using frontend, backend, and database integration.

---

## 🏗 Project Structure

This project is divided into three main parts:

### 1. Frontend
- Built using React
- Handles UI and navigation
- Pages included:
  - Home
  - Product
  - Pricing
  - Support
  - Open Account

### 2. Dashboard
- Separate React application
- Displays stock data visually
- Uses charts for better understanding

### 3. Backend
- Built with Node.js and Express.js
- Handles API requests and database interaction
- Connects to MongoDB using Mongoose

## ✨ Features

- 📊 Dashboard for stock holdings and positions
- 🗄 Data stored in MongoDB and fetched dynamically
- 🔄 REST API integration between frontend and backend
- 📈 Data visualization using charts
- 🧩 Component-based architecture using React

---

## ⚙️ How It Works

1. Stock data (holdings & positions) is stored in MongoDB  
2. Backend APIs fetch this data  
3. Frontend and dashboard display the data dynamically  

---

## 🚧 Current Limitations

- Uses static stock data (no real-time API yet)
- Authentication (login/signup) is under development
